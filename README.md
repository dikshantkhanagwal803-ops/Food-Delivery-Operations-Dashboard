# Food-Delivery-Operations-Dashboard

## 📌 Overview
This project analyzes a food delivery dataset to understand order patterns, delivery efficiency, and operational performance.

## 🛠 Tools Used
- SQL (Data Analysis)
- Google Sheets (Data Cleaning)
- Tableau (Dashboard)

## 🔍 Key Insights
- Peak demand occurs during evening hours
- Delivery time increases with higher demand-supply ratio
- Certain categories have consistently higher delivery times

## 📁 Files in Repository
- Food_Delivery_Dashboard.twbx → Tableau dashboard
- food_delivery_cleaned.csv → Clean dataset
- analysis.sql → SQL queries

## 🚀 How to Use
1. Open `.twbx` file in Tableau
2. Explore interactive dashboard

## 📌 Note
Dataset represents a generic food delivery platform and is used for analysis purposes only.
